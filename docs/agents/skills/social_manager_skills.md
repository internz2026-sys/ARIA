|---------|-------|

|---------|-------|

## Hashtag Strategy

| Platform | Optimal # | Notes |
|----------|-----------|-------|
| Twitter/X | 1-2 | Trending topics only. 2+ reduces engagement |
| LinkedIn | 3-5 | Algorithm sweet spot. 1 broad + 2 niche + 1 branded |
| Facebook | 1-3 | Minimal impact. Use sparingly |

**Selection:** 1 broad industry + 1-2 niche + 1 branded tag. Rotate sets. Caption quality > hashtag volume.

## Twitter/X Thread Template

```
Tweet 1/N: [HOOK — provocative stat, bold claim, or question]
  Signal: "Thread:" or "1/"
  Must stand alone as a compelling tweet

Tweet 2/N: [CONTEXT — set up the problem]

Tweets 3 to N-1: [VALUE — one idea per tweet]
  - Each tweet works if retweeted independently
  - Use line breaks for readability
  - Include data points, examples, mini-stories
  - 200-250 chars per tweet

Tweet N: [CTA]
  - "Follow for more [topic]"
  - "RT tweet 1 if useful"
  - "Try it: [link]"
  - "What would you add?"
```

Threads get 3x more engagement. Replies weighted 75x more than likes.

## LinkedIn Post Templates

### Story Post (highest engagement)
```
[Bold opening — max 210 chars before "See more"]

[2-3 sentence story setup]

[3-5 short paragraphs with insight]
- Bullet points or numbered lists
- One idea per paragraph
- Short sentences

[1-2 sentence takeaway]

[CTA: Question to drive comments]

#hashtag1 #hashtag2 #hashtag3
```

### Listicle Post
```
[Number] things I learned about [topic] after [credibility marker]:

1. [Point] — [1 sentence]
2. [Point] — [1 sentence]
3. [Point] — [1 sentence]

Which resonates most? Drop a number below.
```

**LinkedIn Rules:**
- Short paragraphs (1-3 sentences), blank lines between all
- Carousel posts: 303% more engagement than single images
- Never start with a link (algorithm penalizes)
- Put links in first comment instead
- First 210 chars = your headline
