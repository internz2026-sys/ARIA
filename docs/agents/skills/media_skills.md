# Media Designer Skills

## Image Prompt Engineering

### Prompt Structure
```
[SUBJECT]: What the image shows
[STYLE]: Visual style (photorealistic, flat, minimalist, 3D render)
[COMPOSITION]: Camera angle, framing, layout
[COLORS]: Color palette, mood lighting
[CONTEXT]: Where it will be used
```

### Example Prompts

**Social Media Post:**
```
A clean, minimalist workspace with a laptop showing a SaaS dashboard,
soft natural lighting, muted blue and white color palette,
shot from a 45-degree overhead angle, professional and modern feel.
```

**Ad Creative:**
```
A developer smiling while looking at code on a monitor,
photorealistic style, warm lighting with a slight bokeh background,
brand colors blue and white, close-up portrait composition,
conveys productivity and satisfaction.
```

**Blog Header:**
```
Abstract geometric pattern representing data flow and connectivity,
flat design with gradient blue-to-purple colors, wide horizontal format,
clean and professional, suitable as a blog post header image.
```

**Product Mockup:**
```
A floating laptop mockup displaying a clean web application interface,
3D render style, soft shadow beneath, gradient background
in light gray to white, centered composition, premium feel.
```

## Style Guide by Platform

| Platform | Style | Mood | Colors |
|----------|-------|------|--------|
| LinkedIn | Professional, clean | Trustworthy, authoritative | Blues, grays, whites |
| X/Twitter | Bold, eye-catching | Energetic, conversational | High contrast, brand colors |
| Facebook | Warm, relatable | Friendly, approachable | Warm tones, brand colors |
| Blog | Informative, polished | Educational, modern | Muted, professional palette |
| Ads | Attention-grabbing | Urgent, aspirational | Brand colors with high contrast |

## Quality Checklist
- No text baked into images (add text via design tools later)
- Brand colors referenced when available
- Appropriate aspect ratio for target platform
- Clean composition without visual clutter
- Professional quality suitable for marketing use
- No copyrighted characters, logos, or trademarked imagery

## Integration with Other Agents
- **Content Writer**: Generates blog headers matching article topics
- **Social Manager**: Creates platform-specific post images
- **Ad Strategist**: Produces ad creative assets matching campaign briefs
- **Email Marketer**: Creates hero images for email campaigns
- **CEO**: Receives visual content requests and delegates with context
